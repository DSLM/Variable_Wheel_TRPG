---------------
【初始标签-职业分类】
---------------
[野外生存专家]：角色具备野外判定方向、生火技巧、求生工具使用、野外寻找水源、判定部分物种是否可食用、跟踪动物等技巧，以及角色在危险处境下可以激发自己的生存本能，在没有食物来源的情况下，为了更久的存活甚至可以做到“自产自消”，以及在面临凶猛野兽的情况下可以出色地进行假死。
---------------
[侦探]：角色具有清晰的思路逻辑，在进行思考时很难被情绪或各种感性因素打扰。在具有一定线索的情况下，侦探善于以这些线索为支点进行假设和推理，最终找到合理的真相。
---------------
[商人]：角色因为一贯行商，在经营方面具有很大的技巧，往往可以很好地打理自己的财务。因为经商原因，商人也具备着其他职业很难锻炼出来的机遇性直觉。
---------------
[学生]：因为生活环境问题，学生职业的角色具有较强的学习能力，在遇到问题时可以迅速的用所学知识来思考。也许单方面的阅历不如其他职业，但是学生的综合知识能力却是十分出色的。
---------------
[黑客]：角色分为黑客和白客，两者都拥有较好的电脑水平，黑客大都是非法入侵他人的计算机用于犯罪，也有小部分只是兴趣爱好，黑客大部分都是无组织的。而白客大都是由特殊部门或银行或政府聘请来进行计算机安全维护以及调试的，白客大部分都是有组织有纪律的。
---------------
[军人]：角色组织纪律观念强，通常拥有较强的责任感和荣誉感，执行力强，拥有着良好的心理素质。因为长时间在部队中锻炼的原因，军人角色的身体素质往往会高于其他职业角色。
---------------
[艺人]：角色因为职业所致，所以表演能力强，相比其他职业来讲，艺人们更容易用自己的表演来打动其他人。部分歌手方面的艺人甚至还有可以模仿其他人声音的技巧，以及一些特殊艺人甚至有腹语等常人无法做到的技巧。
---------------
[赌徒]：角色拥有相当的赌徒心理，喜欢当投机者，敢于孤注一掷，却不喜欢思考当前的处境，在赌徒的预期中的目标如果没有出现，那么赌徒就有可能失去理智的用自己的一切进行豪赌，当然这并不是全部赌徒的统一梗概，但无一例外的是，以赌徒作为职业的角色往往心理承受能力高于常人。
---------------
[杀手]：角色是靠接受委托杀人的职业，所以拥有着良好的杀人技巧或潜入技巧，较善于逃脱隐藏自己，具有很多危险性极高的攻击技巧，可能善用枪支、刀具、机关等道具，具体视情况而定。
---------------
[雇佣兵]：角色是为了利益而参加一场武装冲突的人，可以为了金钱参加任何的战争，通常情况下，只要出价够他们可以受雇于任何人。虽然他们身体素质与军人不相上下，但是职业素养上面却有着天差地别。
---------------
[教师]：角色拥有着良好的知识储备和大量的教学经验，善于对他人传授知识。
---------------
[医生]：角色有着良好的医学理论知识，作为医生，他们也具有很强的医学实践经验。
---------------
[运动员]：角色拥有着因长时间锻炼获得的强健体魄，拥有良好的体力，在自己的专向运动方面拥有着很大的优势。
---------------
[作家]：角色是以文化创作为业，写作的人，拥有着良好的文学功底，根据写书的内容拥有着不同的思路和见解，作家们通常都有着天马行空的想象力和逻辑能力。
---------------
[科学家]：角色是对真实自然及未知生命、环境、现象及其相关现象统一性的客观数字化重现与认识、探索、实践的人，进行这各种科学研究或发明，有着极高的知识素养和极高的智商。
---------------
[警察]：角色是一名警察，以遵守着制度，维护社会秩序，坚守岗位，帮助人民群众为己任，作为警察的角色往往都具有很强的正义感。
---------------
[设计师]：在某个特定的专门领域创造或提供创意的工作，从事艺术与商业结合在一起的，根据不同的领域有着不同的专业知识，但大都有良好的空间思维能力。
---------------
[猎人]：角色是从事打猎职业的人，有着良好的狩猎技巧，懂得如何分辨动物的脚印，以及猎杀动物和伪装以及荒野求生的技巧。
---------------
[饲养员]：角色是给负责饲养动物园或者水族馆动物的工作人员，有着对动物较高的亲和力，较懂得如何获取动物的信任以及一些对动物的护理手段。
---------------
[流浪汉]：角色是没有工作和收入，居无定所的人，经常在公园、天桥底、住宅楼梯睡觉和休息，大都靠一些临时工作和乞讨为生。
---------------
[不定工作者]：角色是没有固定工作的人，有着良好的能力，却经常换着工作，做着一些短期的兼职，来共自己生存的人。
---------------
[御宅族]：角色是热衷于亚文化，并对该文化有极度深入的了解的人，例如动漫或是军事此类，对着自己喜欢的文化有着超乎想象的了解以及深入性，还有对热爱文化永不停息的求知欲和了解性。
---------------
[白领]：角色是有着较高教育背景和工作经验的人士，有着固定工作，坐在办公室工作，过着朝九晚五的日子，领着固定的工资的人群。
---------------
[赛车手]：角色是以赛车为职业的人，大都是摩托车手或是F1赛车手，喜欢竞争，角色有着强大的心理素质和较强的身体素质，以及良好的车技。
---------------
[罪犯]：角色曾是因犯罪而服刑役的人，通常是因为一些罪恶之事而进入监狱，大都对社会或是一些人或事物抱有报复心理。
---------------
[主持人]：角色是电视台或是一些节目的主持人，具有采、编、播、控等多种业务能力，良好的台词功底和记忆力。
---------------
[间谍]：角色的职业是通过采取非法或合法手段、通过秘密或公开途径窃取情报或是破坏工作，大都为一些公司的商业机密或者国家机密为行动目标，角色大都是为政府或是一些公司效力。
---------------
[极限运动爱好者]：角色是世界上最勇敢的那一群人，有着顶级的心理素质，角色追求着最大限度地发挥自我身心潜能，向自身挑战，用极限运动来寻求刺激。所有极限运动都是有死亡的危险，面对死亡就是直面恐惧，而角色们就是用着自己的的训练和锻炼来减少恐惧以及死亡的可能。其中有着跑酷、翼装飞行、徒手攀岩、高空跳水等包括但不限于这些的运动。
---------------
[富家子弟]：角色是拥有优秀家庭背景的一群人，从小接受顶尖的教育，活跃在上流社会中，享受世界顶端的资源，这群人有丰富的阅历，以及不俗的艺术欣赏能力，对金钱、物质的诱惑有一定的抵抗能力。
---------------
[宗教职业者]：角色是以某种宗教信仰或某类宗教信仰者服务为职业的人。通常，此类人员自身也信奉某种宗教信仰，并遵守宗教设立的规范或戒律等事项。此类人员大多对广义性质的宗教神学具有一定的理解。
---------------
[神秘主义者]：此类角色曾经生活在魔法和灵能等特殊力量运用度低下的时代，但他们偏偏有极度信奉、渴望了解这一类知识。因而他们在接触到和自己价值观极度不符的东西时会有超乎常人的理解和吸收能力。
---------------
[异能剥离者]：泛指来到变量之轮之前习惯于凭借灵感、魔法、超能力等唯心的力量达成目的的人群。而即使在变量之轮的序列化同调时失去了原本的能力，角色也依然保留着对自己曾拥有过的异能的了解和偏爱。
---------------
[探索者]：角色由于个人的兴趣，长期自愿探索对于原本世界的生命来说未知的领地。他们通常都对“世界之外”抱有极大的好奇心，也甘愿为此身陷险境。由于从事相关的工作，他们大多受过不同程度的身体训练，使得他们具有了很好的身体素质以及独立面对突发状况的能力。常常是勇气与智慧并存的那一位。这类职业可以指代“宇航员/潜水员/自然考察者”等群体。
---------------
[海盗]：指角色经常伙同他人凭依在某种交通工具上，四处发动违法袭击抢劫钱财。这类角色通常残暴没有人性，眼中只有对着目标财宝的贪婪。永远不要试图用利益之外的东西打动一个海盗...当然，也有可能会有相反的个例。
---------------
[窃贼]：常年生活在阴影之中，不愿从事正规工作，也不敢正面劫财劫色的矛盾存在。对于如何隐藏自己颇有心得，也懂得如何察言观色去判断一个人的生活习惯和职业。
---------------
[侍者]：通常泛指有过大量服务业经历的角色。这一类人拥有丰富的社会经历，懂得察言观色，如何与社会上各个阶层的人交流。他们通常能在极端愤怒的情况下保有理性，把道理用委婉的方式表述出来。
---------------
[厨师]：以烹饪为职业、以烹制菜点为主要工作内容的人。作为一名合格的厨师，要有精细的刀工，对火候的掌握要得当，调味要准确适口。要有较好的文化知识素养。要掌握现代营养、卫生等有关烹饪科学方面的基础理论知识，要了解祖国的烹饪文化历史，要懂得一定的民俗礼仪知识。要有一定的美学修养和艺术创新基础。此外，还应该有勇于创造、大胆革新的精神。
---------------
[革命者]：这类角色在来到变量之轮前通常有变革陈腐社会制度，创造更加优越良好的社会环境与政治制度、体制的经历。他们或许并非优秀的政客，但一定具备着学习新思潮、新可能的能力与意愿。后人会为他们的功绩而垂泪，也会因他们的功绩而幸福。此类角色的性格往往具备着极强的社会性与利他性，即使已经来到了变量之轮，这份渴求为普罗大众创造幸福、勇于为无辜弱者做出牺牲的正义与先进思想，也仍然长存心中。
---------------
[武者]：泛指来到变量之轮前习惯于凭借各种武术，达成强身、伤人、明悟等一系列目的的人群。无论是粗通拳脚的三流武者，还是一证永证的彼岸武圣，都可以被归类于此。过往的辉煌与强大，已经尽数伴随序列化同调的过程化为泡影，但武者们曾经思悟的武道，仍然可以让他们有效地完成各种漂亮高效的武术动作，部分寄情于武的武痴，甚至会因此再次走上习武之道。
---------------
[方外之人]：泛指来到变量之轮前便有出世修行，或是以其他方式接触修行蜕凡之道的人群。方外，指的就是世俗之外，尘世之外。他们往往比寻常的角色更加具备淡泊与坚定的品质，也对一些常人眼中神秘莫测的方外之事分外了解。虽然这些经验与知识在变量之轮与任务世界中或许已非其所认识的真理，但在某些时刻或许仍能提供些微帮助。无论是阴阳先生，行游托钵，还是得道大能，大都会被归类为方外之人。从前修炼中的感悟与一身修为，已经被序列化同调化为归尘之物，但一颗向道之心，或许仍能让他们在变量之轮之中，开始第二段生涯。
---------------
[艺术家]：艺术家是指具有较高的审美能力和娴熟的创造技巧并从事艺术创作而有一定成就的艺术工作者。艺术家具有敏锐的感受、丰富的情感和生动的想象能力，同时还掌握一部分艺术创作能力。他们的特点是包括超出常人的丰富的情感，异于常人的审美感受力，强于常人的艺术想象力。
---------------
[政治家]：政治家是指那些在长期的政治实践中涌现出来的具有一定政治远见和政治才干、掌握着政党或国家的领导权力并对社会历史发展起着重大影响作用的领导人物。同时也包裹了政客这个别称，总体来说所有政党引导者都可以以政治家进行概括。
---------------
[心理学家]：心理学家是指从事心理科学专业并有成就的人。随着心理学研究领域的日益多样化，其队伍日益壮大。大多数心理学家在大专院校从事教学和科研工作，他们用于两项工作的时间比例，随不同环境而异。
---------------
[记者]：记者是报社、通讯社、广播电台、电视台等媒体中中担任采访报道摄影的工作人员。一般而言，记者主要在报社、电视台、广播电台、杂志社、通讯社等机构工作。网络出现之后，也有部分记者替网络媒体（网络报）或专业网站工作。所有记者掌握着信息收集，文字编辑等能力。
---------------
[工人]：角色是挣工资而被雇用从事体力或技术劳动的一类人，一般会对一种技术有所专精。
---------------
[农民]：角色是长时期从事农业生产的人，一般会拥有一些农业知识。
---------------

---------------
【初始标签-通用分类】
---------------
[纵观全局]：角色具有很大的宏观概念，通常具备这种标签的人在生前都属于领导者阶级，他们不会拘泥于一些小的利害关系，而是更善于总揽全局来把握更大的利益。
---------------
[领导能力]：通常具备这种标签的人在生前都属于领导者阶级，他们很善于带动其他人去做某些事情，也很善于规划自己的行动方针。
---------------
[普普通通]：实际上普通也算是一种特殊，有些人很善于让自己掩没在人群之中，也很善于让自己变得“大众化”，在某些特定的情况来看，这其实也不算是个坏事。
---------------
[印象淡然]：这通常来源于那些存在感很低的人，他们很难给别人留下什么印象，在别人的生命中，他们也只是一个匆匆过客而已。
---------------
[莽夫]：这通常是那些拥有强大武力的人，拥有强大的力量却不喜欢动脑子，遇到问题时不会变通，和别人交谈更喜欢使用武力威慑。
---------------
[懦夫]：拥有这类的标签的大都是生前软弱无能的人，通常没有自己的主见，比较喜欢从于大众，他们没有干劲，对自己没有信心，性格都较为自卑。
---------------
[交际能手]：这些通常来源于那些性格谦虚待人真诚的人，他们热爱交际，喜欢和别人交朋友，面对陌生人时他们热情，善于用自己的情绪来带动对方。
---------------
[沉默寡言]：拥有此类标签的角色经常不声不响，很少说话，但是在关键时常常语出惊人。
---------------
[自我中心]：角色性格较为自负，自傲，不喜欢听取别人的意见，认为自己才是最好的那个，思考问题时总是以自己的角度为出发点，以自己的实际利益为优先，很少拥有朋友。
---------------
[行动派]：拥有此类的标签的人相较于交谈，他们更喜欢用实际行动来表明自己的态度，有什么想法之后，就回去马上去执行，有着强大的执行力。
---------------
[孤僻]：拥有此标签的人性情孤独，不合群，不能与人保持正常关系，朋友较少，喜欢独来独往，对他人怀有厌烦、戒备和鄙视心理；凡事与己无关、漠不关心，如果和人交流也会缺少热情和活力，显得漫不经心、敷衍了事。
---------------
[穷凶极恶]：拥有的此类标签的角色性格极端残暴凶恶，拥有的标签的人角色大都为罪犯又或者长期参与战争的雇佣兵。
---------------
[异想天开]：拥有此类标签的角色有着超强的想象力，和跳跃性的思维方式，他们的脑洞极大，常常会想到一些荒唐离奇的方法。
---------------
[狡诈]：拥有此类标签的角色阴险狡猾，喜欢算计其他人来获得好处，行为极为小心，不会轻易暴露自己。
---------------
[鬼鬼祟祟]：角色不愿暴露在人前，很多事情都喜欢拐弯抹角的去达成。尽可能去避免一对一的交谈、对决一类行为。不过这并不一定是出于角色的恶意，有时只是不善于交际。
---------------
[爱慕虚荣]：这类角色通常极其关注周围人对自己的评价，把声誉的上升当做最大的目标。有时是不顾别人安慰的自私者，但也可能会为了别人的一句赞赏抛弃自己的性命
---------------
[踏实]：他们通常都喜欢脚踏实地，不愿意在事情还没开始的时候定制一个过于宏达的目标，而是会劝人先从手边的小事做起。行事风格谨慎认真，是合作中极为可靠的存在。
---------------
[守矩]：角色是社群当中最为遵守规矩的那一类，总是希望在尽可能不破坏现有规则的情况下达成目标。具有极高的自律性，只是有时会比较死板难以说动。
---------------
[偷懒耍滑]：这类角色通常让人欲哭无泪。他们懒惰且聪明，总会在行动中试图付出最少的努力来获得和别人一样的成果。
---------------
[正大光明]：这类角色通常都来自于台面上的职业。他们喜欢堂堂正正通过正规手段解决问题，对不法分子和邪恶的性格有着先天的反感。甚至必要时刻会出手声张正义。
---------------
[混沌无序]：这类角色依循自己的冲动行事，是完全的个人主义者。非常重视自己的自由权利，并不致力于守护别人的自由。他们蔑视权 威，愤恨约束并且挑战传统。会做出任何因为自己的欲望、恨意和破坏欲而驱使自己做出的事。当然，往往也意味着他们的行为是不可被估量的，不一定是邪恶，但一定是未知。
---------------
[精神疾病]：角色大脑功能失调，导致认知、情感、意志和行为等精神活动出现不同程度障碍。可能会是应为遗传，也可能是应为角色过去经历过一些不太好的事情。角色可能会突然做出伤害自己的行为，也可能非常健忘，如上一秒吻过你下一秒就忘了你是谁。
---------------
[暴躁]：角色在一定场合受到不利于己的刺激就暴跳如雷，对人对事也可能伴有消极激情的火爆脾气。
